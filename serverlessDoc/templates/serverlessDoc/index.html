Serverless Function api endpoint: "To Send Whatsapp Messages"

Note:

All endpoint url are valid

Introduction

This Documentation contains a Serverless function written in Python. to send an Messages using your Token and PhonenumbersID. Serverless Function component

Requirements

  • You need a facebook developer account. If you don't already have one,
  • The phone number you are sending a message to also has to be a Whatsapp verified phone number.
  • You need to add your facebook developer TOKEN and FROM_PHONE_NUMBER_ID to the connect to the Functions API.
  • (Httpie) is a command-line HTTP client. Its goal is to make CLI interaction with web services as human-friendly as possible. HTTPie is designed for testing, debugging, and generally interacting with APIs & HTTP servers.
  • Optional python Request libery
  • Curl client

    sending Template Message........

    Required Parameter

  • Token
  • FromPhoneNumberID
  • PhoneNumber
  • Template
  • Days

    • To send Message using Httpie

    https https://faas-fra1-afec6ce7.doserverless.co/api/v1/web/fn-ca632dbd-84ac-49ad-87a7-9c107cbf0c54/messages/custom Token==token FromPhoneNumberID==FromPhoneNumberID PhoneNumber==PhoneNumber Templace==sample_shipping_confirmation Days==23


    To send Message using curl

    curl -X POST -H 'Content-Type: application/json' {your-url} -d '{"Token":"iuoueihuy7y3272980912","FromPhoneNumberID":"123456789012435", "PhoneNumber":"+2348120385367", "Template":"sample_shipping_confirmation","Days":"23"}'

    Response

    { "Status": "message sent" }
